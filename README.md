# refiner

CSV stream library for manipulating data

# Install

	$ npm install


# Test

Run all test cases

	$ mocha 

Run a particular test case (-g)

	$ mocha -g head
	$ mocha -g sunrise

# Resources

- BDD-style using [should](http://chaijs.com/api/bdd/)
- Assertion library for streams [stream-assert][https://www.npmjs.com/package/stream-assert]
